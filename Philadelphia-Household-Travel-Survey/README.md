# Philadelphia Household Travel Survey

This repository is one module in the larger Philadelphia Household Travel Survey project. The full project is organized at the GitHub Project level so the related repositories can be managed together while the individual repositories remain hidden.

## Project Scope

The project is based on the 2012 Philadelphia Household Travel Survey and related transit datasets. It studies household travel behavior, mode choice, income and vehicle ownership, daily activity patterns, transportation modeling, transit performance metrics, and project evaluation methods.

## Project Modules

| Module | Repository role | Scope |
| --- | --- | --- |
| Core Survey Analysis | `Philadelphia-Household-Travel-Survey` | Household travel behavior, mode choice, income, car ownership, and daily trip patterns |
| Transportation Modeling | `Philadelphia-Household-Travel-Survey-Transportation-Modeling` | Trip distributions, non-travelers, demographic patterns, and transit ridership modeling |
| Transit Data and Metrics | `Philadelphia-Household-Travel-Survey-US-Transit-Data-and-Metrics` | U.S. transit system performance, fare recovery, passenger miles, speeds, and operating metrics |
| Project Evaluation | `Philadelphia-Household-Travel-Survey-Project-Evaluation` | Cost-benefit analysis, NPV scenarios, unquantified impacts, and evaluation framing |

## Suggested Reading Order

1. Core Survey Analysis
2. Transportation Modeling

4. Project Evaluation

## GitHub Project Setup

The GitHub Project should use these fields:

| Field | Suggested values |
| --- | --- |
| Status | Backlog, Ready, In Progress, Review, Done |
| Workstream | Survey Analysis, Transportation Modeling, Transit Metrics, Project Evaluation |
| Priority | High, Medium, Low |
| Output Type | Code, Report, Dataset, Visualization, Presentation |
| Target Date | Date field |
| Owner | Assignee field |

Recommended views:

| View | Grouping or filter |
| --- | --- |
| Overview | Group by Workstream |
| Workstream Board | Board grouped by Status |
| Research Outputs | Filter by Output Type |
| Modeling | Filter Workstream = Transportation Modeling |
| Evaluation | Filter Workstream = Project Evaluation |
| Transit Metrics | Filter Workstream = Transit Metrics |
