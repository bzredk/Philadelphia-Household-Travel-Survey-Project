# Philadelphia Household Travel Survey Project

This repository is the public entry point for the Philadelphia Household Travel Survey work. It organizes four private module repositories into one readable project structure for portfolio, review, and navigation purposes.

The source repositories for the modules remain private. This public repo describes what each module does and how the pieces fit together without exposing private source files, notebooks, or datasets.

## Project Structure

| Module | Focus | Private source |
| --- | --- | --- |
| [Core Survey Analysis](modules/core-survey-analysis/) | Household travel survey preparation, exploratory analysis, and core travel behavior metrics. | Private module repository |
| [Project Evaluation](modules/project-evaluation/) | Evaluation methods, project impact framing, benefits, costs, and decision metrics. | Private module repository |
| [Transportation Modeling](modules/transportation-modeling/) | Transportation modeling workflows, mode/trip analysis, and scenario modeling. | Private module repository |
| [U.S. Transit Data and Metrics](modules/us-transit-data-and-metrics/) | Transit data collection, U.S. transit indicators, and comparative transit metrics. | Private module repository |

## Suggested Reading Path

1. Start with Core Survey Analysis to understand the household travel survey foundation.
2. Read U.S. Transit Data and Metrics for the external transit context and benchmark metrics.
3. Review Transportation Modeling for the modeling workflow built on the survey and transit inputs.
4. Finish with Project Evaluation for the final assessment and interpretation layer.

## Repository Purpose

This repo is intentionally lightweight. It is meant to act as the public project layer while the working code and detailed data remain in private repositories.
