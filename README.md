# Philadelphia Household Travel Survey Project

This repository is the public monorepo for the Philadelphia Household Travel Survey work. It brings the contents of four related project repositories into one organized place.

The original four repositories remain private, but their current project files are copied here as subdirectories so the work can be viewed from one repository.

## Project Structure

| Subproject | Focus |
| --- | --- |
| [Philadelphia-Household-Travel-Survey](Philadelphia-Household-Travel-Survey/) | Household travel survey preparation, exploratory analysis, and core travel behavior metrics. |
| [Philadelphia-Household-Travel-Survey-Project-Evaluation](Philadelphia-Household-Travel-Survey-Project-Evaluation/) | Evaluation methods, project impact framing, benefits, costs, and decision metrics. |
| [Philadelphia-Household-Travel-Survey-Transportation-Modeling](Philadelphia-Household-Travel-Survey-Transportation-Modeling/) | Transportation modeling workflows, mode/trip analysis, and scenario modeling. |
| [Philadelphia-Household-Travel-Survey-US-Transit-Data-and-Metrics](Philadelphia-Household-Travel-Survey-US-Transit-Data-and-Metrics/) | Transit data collection, U.S. transit indicators, and comparative transit metrics. |

## Suggested Reading Path

1. Start with `Philadelphia-Household-Travel-Survey` to understand the household travel survey foundation.
2. Read `Philadelphia-Household-Travel-Survey-US-Transit-Data-and-Metrics` for the external transit context and benchmark metrics.
3. Review `Philadelphia-Household-Travel-Survey-Transportation-Modeling` for the modeling workflow built on the survey and transit inputs.
4. Finish with `Philadelphia-Household-Travel-Survey-Project-Evaluation` for the final assessment and interpretation layer.

## Repository Layout

Each subproject directory contains the working files copied from its original repository, including code, data files, reports, and supporting documents.
