# U.S. Transit Data and Metrics

This module represents the private repository focused on U.S. transit data and comparative transit indicators.

## Scope

- Collect, clean, and organize transit data sources.
- Build indicators such as ridership, service, cost, passenger miles, and related transit metrics.
- Provide benchmark context for the Philadelphia household travel survey project.

## Role in the Project

U.S. Transit Data and Metrics supplies the external transit context used by the modeling and evaluation modules.

## Source

The working source repository for this module is private.
