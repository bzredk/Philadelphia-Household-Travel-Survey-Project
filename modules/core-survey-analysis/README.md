# Core Survey Analysis

This module represents the private repository focused on the household travel survey foundation.

## Scope

- Prepare and clean household travel survey records.
- Explore household, person, trip, and mode-level patterns.
- Produce core travel behavior metrics used by downstream modules.

## Role in the Project

Core Survey Analysis is the base layer. The transportation modeling and evaluation modules depend on the cleaned survey structure and baseline findings from this work.

## Source

The working source repository for this module is private.
